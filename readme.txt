== Description ==

This block allows you to create responsive, customizable carousels using WordPress cover blocks as slides. It was initially built during the [Speed Build Challenge](https://asia.wordcamp.org/2025/session/wordpress-speed-build-challenge/) at WordCamp Asia 2025.

**Features:**

* Display 1-3 slides at a time
* Adjustable gap between slides
* Customizable minimum height
* Optional infinite loop functionality
* Interactive floating arrows for navigation
* Color customization through WordPress color palette
* Full support for block alignment (full-width, wide, and default)
* Fully responsive with mobile optimization
* Compatible with WordPress block themes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/carousel-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Carousel Block in any post or page

== Frequently Asked Questions ==

= Can I use any block as a slide? =

Currently, the carousel is designed to work specifically with Cover blocks as slides.

= Is it responsive? =

Yes! The carousel automatically adjusts to one slide on mobile devices.
